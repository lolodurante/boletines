# Modulo director

El director accede a dashboard completo, configuracion academica, revision de boletines y estadisticas.

## Alcance inicial

- Resumen general.
- Avance por curso y periodo.
- Boletines listos.
- Alertas por email faltante.
- Docentes pendientes.
- Distribucion de calificaciones.
- Historial filtrable.

## Reglas

- Solo `DIRECTOR` y `ADMIN` revisan o envian boletines.
- Solicitar correccion cambia el boletin/evaluacion a estado de revision.
- El director puede agregar observacion institucional antes del envio.
