# Modulo docente

El docente ve cursos, materias y alumnos asignados. No puede configurar periodos ni generar boletines.

## Alcance inicial

- Lista de cursos asignados.
- Lista de alumnos.
- Formulario de evaluacion por criterio.
- Guardado parcial.
- Envio final.
- Observaciones por materia y generales.
- Banner de fecha limite.

## Reglas

- Solo puede editar alumnos asignados.
- El envio final exige todos los criterios requeridos.
- Los borradores no desbloquean boletines para direccion.
